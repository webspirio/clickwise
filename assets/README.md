# Clickwise WordPress Plugin Assets

This directory contains all the visual assets for the Clickwise WordPress plugin, designed for WordPress Plugin Directory submission and general branding purposes.

## WordPress Plugin Directory Assets

### Plugin Icons
- **`icon-128x128.svg`** - Standard WordPress plugin icon (128×128px)
- **`icon-256x256.svg`** - High-resolution WordPress plugin icon (256×256px)

*These files should be converted to PNG format for WordPress.org submission*

### Plugin Banners
- **`banner-1544x500.svg`** - High-resolution plugin directory banner (1544×500px)
- **`banner-772x250.svg`** - Standard plugin directory banner (772×250px)

*These files should be converted to PNG or JPG format for WordPress.org submission*

## Branding Assets

### Logos
- **`clickwise-logo.svg`** - Main Clickwise logo (original design)
- **`webspirio-logo-24x24.svg`** - Webspirio developer logo (24×24px)

### Visual Elements
- **Main Logo**: Organic/rounded shapes from original Clickwise design

- **Icons**: Clean, minimal design with WordPress styling
- **Patterns**: Subtle dot overlays for texture

## Conversion Notes for WordPress.org

For WordPress Plugin Directory submission, convert these SVG files to:

1. **Icons**: PNG format, exact pixel dimensions (128×128, 256×256)
2. **Banners**: PNG or JPG format, exact pixel dimensions (772×250, 1544×500)

### Recommended Conversion Settings:
- **Format**: PNG (for transparency) or high-quality JPG
- **Color Space**: sRGB
- **Resolution**: 72 DPI (web standard)
- **Compression**: Optimized for web (balance quality vs file size)

## Usage Guidelines

### Plugin Directory
- Use `icon-*.svg` files for plugin listings
- Use `banner-*.svg` files for plugin page headers
- Maintain aspect ratios and exact dimensions

### Documentation & Marketing
- Use `clickwise-logo.svg` for primary branding
- Include Webspirio attribution where appropriate
- Maintain color scheme consistency

### Development
- SVG format preferred for scalability
- All assets are vector-based for crisp display at any size
- Optimized for both light and dark backgrounds
